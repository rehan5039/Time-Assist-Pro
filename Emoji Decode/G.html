How to Use Emoji

You can use these emoji by simply copying and pasting them into the editor on any social media site, or anywhere else you'd like to use them.

Emoji are shown differently on various devices and platforms. For instance, they could look somewhat rounded on Twitter and wavy on Apple devices and Facebook. On Windows, some country flags might not function and might only show two-letter country codes.

All in all, though, most of the emoji on this list continue to function across platforms.

How to Use the Unicode Characters

Unicode characters are universal encoding standards that assign a number to every character, emoji, or symbol.

To insert any of the emojis in HTML, for example, remove the U+ that all the characters start with, type in the hexadecimal character reference, add an ampersand (&), number sign (#) and x at the front, and a semi-colon at the end.
